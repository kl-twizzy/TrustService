# Концепция дипломного проекта

## Тема

Разработка сервиса для анализа репутации продавцов в сфере электронной коммерции.

## Проблема

Пользователь маркетплейса часто видит высокий рейтинг товара, но не может быстро понять:

- насколько надежен сам продавец;
- не накручен ли рейтинг;
- нет ли подозрительных отзывов;
- не ухудшилось ли качество обслуживания в последнее время.

## Решение

Предлагается сервис "Карта надежности продавцов маркетплейсов", который в виде браузерного расширения отображает:

- рейтинг доверия продавцу;
- достоверность рейтинга карточки товара;
- предупреждения о возможной накрутке;
- краткое объяснение результата.

## Практическая ценность

- снижение риска неудачной покупки;
- повышение прозрачности выбора продавца;
- возможность масштабирования под разные маркетплейсы;
- простая интеграция за счет формата браузерного расширения.
