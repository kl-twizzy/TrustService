# Browser Extension

## Как использовать

1. Откройте `chrome://extensions/`
2. Включите режим разработчика
3. Нажмите `Load unpacked`
4. Выберите папку `browser-extension`
5. Откройте карточку товара на OZON, Wildberries или Яндекс Маркете
6. Нажмите на иконку расширения
7. Нажмите кнопку `Проверить товар`

## Как работает

Расширение:

- берет URL текущей вкладки;
- отправляет его в backend;
- получает результат проверки;
- показывает trust score и признаки риска прямо в popup.

Используемый endpoint:

`POST http://localhost:8080/api/v1/trust/analyze-url`
